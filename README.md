# Minified Documentation

Documentation for **Minified**, a Java library for building Minecraft launchers.

Page: https://dervarex.github.io/minified-docs/

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the documentation:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run Astro CLI commands:

```bash
npm run astro -- <command>
```
