---
title: Mojang Mappings
---
Mojang mappings are the official names provided by Mojang for Minecraft's obfuscated classes, fields and methods.
### Differences to Yarn
- Official Recognition: Uses exactly the same names Mojang uses in their source code
- No parameter names: Mojang mappings generally do not include method parameter names or Javadocs. Yarn on the other hand, includes documentation for a lot of methods
:::note

Looking for Yarn mappings? See **[Yarn](./yarn)**.

:::
