---
title: Fabric API

---

The Fabric API is an essential core library for the Fabric Ecosystem. It provides Essential Hooks and interoperability mechanisms for modding with Fabric.
Like any other library mod, it requires the <a href="./fabric-loader" className="link">Fabric Loader</a> and must be placed in the `mods` folder.

### Examples

- Exposing functionality that is useful but difficult to access for many mods such as particles, biomes and dimensions
- Adding events, hooks and APIs to improve interoperability between mods.
- Essential features such as registry synchronization and adding information to crash reports.
- An advanced rendering API designed for compatibility with optimization mods and graphics overhaul mods.
