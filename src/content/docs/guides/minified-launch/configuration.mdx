## 1. The LaunchConfigurator
The `LaunchConfigurator` is the central configuration object for launching Minecraft with Minified.
It can be built using the `LaunchConfigurator.Builder` class, which provides a fluent API for setting various launch parameters.

Example usage:

<Tabs>
  <TabItem label="Java">

```java
LaunchConfigurator config = new LaunchConfigurator.Builder()
        .downloadThreads(10)
        .launcherName("YourLauncherName")
        .launcherVersion("1.0.0")
        .assetsDirectory(Path.of("<assets-directory>"))
        .librariesDirectory(Path.of("<libraries-directory>"))
        .jarFile(Path.of("<client.jar>"))
        .isDemoUser(false)
        .loader(new VanillaLoader("1.21.11"))
        .build();
);
```

  </TabItem>

  <TabItem label="Kotlin">

```kotlin

    .downloadThreads(10)
    .launcherName("YourLauncherName")

    .librariesDirectory(Path.of("<libraries-directory>"))
    .jarFile(Path.of("<client.jar>"))
    .isDemoUser(false)

    .build()

```
